# Paterson 官网重设计方案

## 1. 设计定位

Paterson 官网建议定位为：

**高级稳重的水漆整家定制招商官网**

核心目标不是单纯展示品牌，而是帮助潜在加盟商快速理解：

- 百得胜 Paterson 是谁
- 为什么水漆整家定制有差异化
- 品牌、供应链、技术和上市公司背景是否可信
- 加盟后能获得什么支持
- 如何快速咨询加盟

整体气质应是成熟、可信、高端、健康、有技术含量，而不是传统促销型招商页。

## 2. 同类型网站参考

### 客来福

参考点：

- 水漆全屋定制定位接近 Paterson
- 大量真实家居空间图
- 产品系列、全屋定制、招商加盟结构清晰

不建议照搬：

- 页面信息偏传统
- 文案和视觉层级较散
- 现代感不足

### 兔宝宝家居

参考点：

- 环保、健康、板材、科技背书强
- 招商表单位置明确
- 品牌实力、技术优势、环保优势表达完整

不建议照搬：

- 页面信息过密
- 绿色环保视觉容易显得常规
- 招商氛围偏传统

### 好莱客

参考点：

- 上市品牌官网的稳重感
- 品牌介绍、产品推荐、招商加盟、投资者关系结构完整
- 适合参考“成熟品牌官网”的信息组织

不建议照搬：

- 页面视觉偏旧
- 缺少强首屏转化
- 产品和招商之间的叙事连接不够强

### 劳卡

参考点：

- 空间场景叙事较完整
- “十大空间”“风格分类”“全屋解决方案”适合 Paterson 借鉴
- 家居生活方式表达较强

不建议照搬：

- 信息量偏大
- 动效和视觉容易显得繁重
- 招商模块可以更聚焦

### Open Design

参考点：

- 留白充足
- 信息节奏清晰
- 模块叙事顺滑
- 首屏主张明确

不建议照搬：

- SaaS / AI 工具风不适合家居品牌
- 黑白极简和科技产品语气不应直接套用

### qiaomu-design

参考点：

- 反模板化设计思路
- 先做多个视觉方向小样再确认
- 避免 AI 味、紫蓝渐变、空洞文案

不建议照搬：

- 它是设计方法论，不是家居官网视觉参考

## 3. 设计风格

### 风格关键词

- 高级稳重
- 健康环保
- 水漆科技
- 整家定制
- 上市品牌背书
- 招商转化
- 真实空间感
- 克制动效

### 整体气质

页面应该像一个成熟家居品牌的招商中枢，而不是促销落地页。

视觉上要做到：

- 高端但不冷淡
- 环保但不廉价
- 招商但不土味
- 科技但不 SaaS 化
- 品牌感强但不过度使用 Logo 色

### 不建议的风格

避免以下方向：

- 大面积绿色环保风
- 大面积红黄品牌色
- 紫蓝渐变 AI 风
- 玻璃拟态科技风
- 过多圆角卡片
- 传统招商长图风
- 密集新闻门户风

## 4. 配色系统

### 主色策略

Paterson Logo 中有红色和黄色，但不建议大面积铺满页面。

推荐使用：

- 暖白作为主背景
- 石墨黑作为主文字
- 深木色建立家居质感
- 香槟金建立高端感
- 品牌红黄只做点缀
- 深健康绿少量用于环保科技模块

### 色彩 Token

| 用途 | 色值 | 说明 |
|---|---:|---|
| 页面主背景 | `#F7F4EE` | 暖白，提升高级感和家居温度 |
| 内容底色 | `#FFFFFF` | 表单、导航、局部内容区 |
| 主文字 | `#24211D` | 深木黑，稳重可信 |
| 辅助文字 | `#6F6A61` | 说明文案、次级信息 |
| 弱文字 | `#A39C90` | 标签、时间、英文小字 |
| 深木色 | `#7A5438` | 标题强调、分区背景、家居质感 |
| 香槟金 | `#B99A63` | CTA、数字、细线、hover |
| 深健康绿 | `#1F3A34` | 水漆、环保、康氧科技模块 |
| 品牌红 | `#C91F2B` | Logo 呼应、小面积重点 |
| 品牌黄 | `#F2B705` | Logo 呼应、局部高亮 |
| 分割线 | `#E5DED2` | 模块边界、表单边框 |
| 深色背景 | `#171512` | 深色招商/科技区块 |

### 色彩比例

推荐比例：

- 暖白 / 白色：60%
- 石墨黑 / 深木黑：20%
- 木色 / 香槟金：15%
- 品牌红黄 / 深健康绿：5%

### 使用原则

- CTA 主按钮使用香槟金或深木色，不直接用大红。
- 品牌红黄只用于图标、小标识、局部强调。
- 环保科技模块可以使用深健康绿，但不要变成“绿色环保模板页”。
- 大面积背景优先使用暖白、深木色或真实空间图。

## 5. 字体系统

### 中文字体

推荐：

```css
font-family: "PingFang SC", "Microsoft YaHei", "Noto Sans CJK SC", sans-serif;
```

### 英文字体

英文只作为辅助标签，不作为主要视觉语言。

推荐：

```css
font-family: "Inter", "Helvetica Neue", Arial, sans-serif;
```

### 字重

| 场景 | 字重 |
|---|---:|
| Hero 主标题 | 500 / 600 |
| 模块标题 | 500 / 600 |
| 正文 | 400 |
| 数据数字 | 600 |
| 按钮 | 500 |

### 字体原则

- 中文标题不要过粗。
- 英文标签可小写或全大写，但不要抢中文主标题。
- 不使用斜体。
- 不使用花哨衬线字体。
- 不做过度字距。
- 长中文标题优先换行，而不是压缩字号。

## 6. 页面结构

### 顶部导航

建议导航：

- 首页
- 品牌实力
- 产品系统
- 水漆科技
- 招商加盟
- 案例门店
- 新闻中心
- 联系我们

右侧固定：

- 招商热线
- 加盟咨询 CTA

### 首页结构

#### 1. Hero 首屏

内容：

- 大幅真实家居空间图
- 主标题：水漆整家定制加盟新增长
- 副标题：德尔未来旗下主营品牌，专注水漆柜墙门系统与环保整家定制
- CTA：立即咨询加盟 / 查看品牌实力
- 关键信任点：2001 年、股票代码 002631、水漆科技、环保整家

视觉：

- 图片 full-bleed
- 文案靠左或底部
- 使用深色半透明遮罩保证可读性
- 不用厚重白色卡片压住图片

#### 2. 品牌信任背书

展示：

- 成立于 2001 年
- 德尔未来旗下
- 股票代码 002631
- 水漆团标主编单位
- 智能涂装线
- 全国门店 / 经销商数据

表现方式：

- 大数字
- 短标签
- 轻动画递增

#### 3. 水漆科技

重点讲：

- 水漆产业链闭环
- 全水性 3D 智能涂装线
- 油改水 4.0 工厂
- 环保健康标准
- 康氧卧房
- 高硬度全水性涂料涂饰技术

视觉：

- 深色背景
- 细线结构图
- 节点式动效
- 局部使用深健康绿和香槟金

#### 4. 整家产品系统

不要只列“衣柜、橱柜、家具”。

建议改成空间系统：

- 玄关系统
- 客餐厅系统
- 卧室系统
- 儿童房系统
- 厨房系统

视觉：

- 真实空间图
- hover 展示产品卖点
- 图片轻微放大
- 文案简短

#### 5. 加盟优势

建议分成六类：

- 品牌优势
- 产品优势
- 供应链优势
- 设计支持
- 营销支持
- 数智化运营支持

每类只放 2-3 行关键内容，避免大段文案。

#### 6. 加盟流程

流程：

1. 在线咨询
2. 总部沟通
3. 实地考察
4. 城市评估
5. 签约合作
6. 建店培训
7. 开业扶持
8. 持续运营

视觉：

- 横向流程线
- 滚动时逐步点亮
- 移动端改为纵向时间线

#### 7. 案例与门店

内容：

- 旗舰店形象
- 优秀加盟商案例
- 终端空间照片
- 城市门店分布

目的：

- 增强加盟可信度
- 展示终端落地能力

#### 8. 新闻与荣誉

只展示高价值内容：

- 技术认证
- 行业奖项
- 展会新闻
- 标准制定
- 品牌大事件

普通资讯不要占太大篇幅。

#### 9. 加盟留资表单

字段：

- 姓名
- 手机号
- 意向城市
- 投资预算
- 备注

旁边展示：

- 招商热线
- 信息保密承诺
- 总部咨询时间

移动端：

- 底部固定“加盟咨询”按钮
- 点击后展开表单或拨打电话

## 7. 动画策略

### 动画原则

动画要提升高级感和阅读引导，不要炫技。

原则：

- 慢
- 轻
- 克制
- 有层次
- 不影响阅读
- 不阻塞页面滚动

### 推荐动画

#### Hero 图片慢推

效果：

- 首屏背景图缓慢放大或平移
- 增加空间沉浸感

参数：

```css
duration: 8000ms-12000ms;
scale: 1.00 -> 1.04;
easing: linear or ease-out;
```

#### 标题淡入上移

效果：

- 主标题、副标题、按钮依次进入

参数：

```css
duration: 400ms-600ms;
translateY: 16px -> 0;
opacity: 0 -> 1;
stagger: 80ms-120ms;
easing: cubic-bezier(0.22, 1, 0.36, 1);
```

#### 导航吸顶

效果：

- 顶部导航从透明过渡到白色
- 加极轻阴影

参数：

```css
duration: 240ms;
background: transparent -> rgba(255,255,255,0.94);
box-shadow: 0 8px 24px rgba(36,33,29,0.06);
```

#### 数字递增

适用：

- 成立年份
- 门店数量
- 专利数量
- 工厂面积

原则：

- 只播放一次
- 进入视口后触发
- 不循环

#### 产品卡片 hover

效果：

- 图片轻微放大
- 遮罩加深
- 文案上浮
- CTA 出现

参数：

```css
duration: 220ms-300ms;
scale: 1 -> 1.035;
```

#### 水漆科技节点动画

效果：

- 产业链节点依次点亮
- 细线从左到右绘制
- 表达技术闭环

适用模块：

- 水漆产业链闭环
- 智能涂装线
- 康氧卧房系统

#### 加盟流程进度线

效果：

- 用户滚动到流程区时，步骤逐个点亮
- 移动端使用纵向时间线

### 禁用动画

不要使用：

- 粒子背景
- 光球漂浮
- 大面积流体渐变
- 文字逐字跳动
- 页面滚动劫持
- 复杂 3D
- 自动弹窗遮挡首屏
- 快速闪烁动效

### 无障碍降级

需要支持：

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation: none;
    transition-duration: 0.01ms;
  }
}
```

## 8. 移动端策略

移动端重点不是压缩桌面版，而是优先保留转化路径。

移动端首屏必须保留：

- Logo
- 主标题
- 加盟咨询 CTA
- 招商热线或拨号入口

移动端调整：

- 导航折叠为菜单
- 产品空间改为横向滑动或单列
- 加盟流程改为纵向时间线
- 留资表单字段减少
- 底部固定 CTA

底部固定栏：

- 加盟咨询
- 拨打电话
- 门店查询

## 10. 验收标准

设计稿或页面完成后，需要检查：

- 首屏是否 3 秒内讲清品牌和招商价值
- 是否能明显看出“水漆整家定制”差异化
- 是否有足够上市公司、技术、工厂、标准背书
- 是否避免了传统招商页的廉价感
- 红黄品牌色是否控制在小面积
- CTA 是否清晰且多处可达
- 移动端是否能快速拨号或提交咨询
- 动画是否克制，不影响阅读
- 图片是否真实、高级、清晰
- 中文长文案是否不溢出、不重叠

## 11. 最终建议

Paterson 最适合的方向不是“互联网科技感”，也不是“传统家居招商长图”，而是：

**用高端家居空间图建立审美，用水漆科技建立差异化，用上市品牌和产业链背书建立信任，用清晰招商路径完成转化。**

设计上应比现有同类家居官网更现代、更克制、更有留白；配色上以暖白、木色、石墨灰、香槟金为主，红黄品牌色只做点缀；动画上使用慢推、淡入、数字递增、节点点亮和流程进度线，形成高级但不过度的动态体验。
